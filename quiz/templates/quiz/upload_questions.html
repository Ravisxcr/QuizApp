JSON Format Guide
Format 1 (Your Format):
  • question - The question text
  • options - Answer choices (for multiple choice)
  • answer - Correct answer
  • difficulty - 1-2 (easy), 3 (medium), 4-5 (hard)
Format 2 (Extended Format):
  • question_text - The question
  • correct_answer - Correct answer
  • options - Multiple choice options (required)
  • explanation - Answer explanation
  • difficulty - easy, medium, hard

Note: All questions are treated as multiple choice. The options field is required and should contain answer choices.

Example (Your Format):
[
  {
    "question": "What is 2+2?",
    "options": {
      "A": "3",
      "B": "4", 
      "C": "5"
    },
    "answer": "B",
    "difficulty": 2
  }
]
